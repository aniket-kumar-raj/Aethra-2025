# Flappy Bird – Crazy Project

This is a fully playable Flappy Bird game built inside....

---

## How to Open and Start the Game

1. Open the `.xlsm` file **using Microsoft Excel only**.  
   Other apps cannot run the game.

2. At the top of Excel, you will see a yellow bar saying:  
   **Security Warning: Macros have been disabled.**

3. Click the **Options…** button next to that warning.

4. Select **Enable this content**.

5. Click **OK**.

6. On the sheet, click the **Start** button located in the bottom-right corner.

7. A 3-second countdown will appear. During the countdown, move your mouse to the **Jump** button.

8. Click **Jump** to make the bird go up.

---

## Controls

- **Jump** button → Makes the bird flap upward  
- Avoid crashing into obstacles

---

## Important Notes

- The game works only in **Microsoft Excel**.  
- You must enable macros every time you open the file.  
- Keep the `flappy.wav` sound file in the **same folder** as the Excel file so the jump sound plays correctly.
